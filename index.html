🚀 Setup Guide

WebContainer Environment Notice

This project requires local Python installation with system-level access for audio processing and browser automation. The current web environment has limitations.

1 Prerequisites

  • Python 3.8+ installed
  • Chrome browser
  • VB-Cable for audio routing
  • Gemini API key

2 Install Dependencies

Terminal
pip install -r requirements.txt

3 Environment Setup

Create a .env file:

.env
GEMINI_API_KEY=your_gemini_api_key_here

4 Run the Bot

Terminal
python run_bot.py --url "https://meet.google.com/your-meeting-id" --role "Your role in the meeting"

🎧 Audio Routing Setup (Critical!)

Windows:

  1. Install VB-Cable from https://vb-audio.com/Cable/
  2. Set 'VB-Cable Input' as TTS output
  3. In Zoom/Meet, set mic to 'VB-Cable Output'
  4. Route system audio to 'Cable B' to hear the meeting
